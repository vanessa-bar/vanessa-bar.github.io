Friendship Troopers

Description du projet

Friendship Troopers est un site web réalisé avec 8 camarades. Il s'agit d'un réseau social qui a pour objectif de pousser les gens à communiquer à travers un système d'énigmes et de points. Il cible les personnes intéressées par la science-fiction et qui aiment jouer.

Compétences développées

J'ai été la graphiste de ce projet. Ainsi, j'ai réalisé les maquettes du site et les éléments graphiques. J'ai également aidé à développer. Pour le front, nous avons utilisé le framework vue.js. Pour le back, nous avons développé une API en php.
