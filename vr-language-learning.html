VR Language Learning

Description du projet

VR Language Learning est une application mobile en réalité virtuelle permettant d'apprendre une langue. Ce projet a été mené durant ma 2e année d'IMAC, en équipe de 4. Nous devions développer un niveau du jeu, se déroulant dans un café parisien et ayant pour objectif d'apprendre le français à un anglophone. Cependant, nous avons choisi de rendre le code le plus modulable possible afin de rendre possible l'apprentissage d'autres langues. Dans une optique d'immerger l'utilisateur le plus possible, nous avons animé les personnages et avons ajouté des bruits d'ambiance.

Compétences développées

Lors de ce projet, j'avais le rôle de développeuse. J'ai ainsi appris à utiliser Unity (et découvert le C#). Afin de créer les scénarios des niveaux, j'ai utilisé du XML. J'ai aussi beaucoup appris sur la réalité virtuelle, puisque je n'en avais jamais fait. J'ai alors découvert que le développement d'applications en réalité virtuelle se pliait à certains principes d'ergonomie, afin de rester agréable et utilisable. Ici, un des vrais challenge était de tout contrôler au regard, l'application étant sur Android.
